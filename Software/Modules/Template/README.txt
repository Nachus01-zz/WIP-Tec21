Software module name meanings:

ci2c: Centrar I2C, I2C communication with the central board.
EUSART: Enhanced Universal Synchronous-Asynchronous Receiver-Transmitter, a variant of the UART serial port (used for debugging).
Interrupts: Interrupt configurations.
li2c: Local I2C, I2C communication within the module board.
Miscelaneous: Functions and routines that do not fit into any other module.
